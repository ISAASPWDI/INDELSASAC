Instrucciones para Formato JSON

Aquí hay algunas reglas básicas para escribir en formato JSON:

  • Use comillas dobles para texto y números.
  • Separe los pares "detalle:caracteristica" con comas.
  • Para el ultimo valor, no use coma.

Ejemplo:


    {

    "Espesor": "Desde 0.10mm hasta 0.50mm",
    "Clase Térmica": "A (105°C)",
    "Formato": "1000 x 1000 mm Aprox.",
    "Tension de ruptura": "3000V" <-- Sin coma

    }
